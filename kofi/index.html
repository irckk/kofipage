Karo Software

Karo Software は、日常のデジタル作業を効率化する Chrome拡張機能 などを個人開発しています。 すべてのプロダクトは 基本機能を無料 で提供し、開発の継続を応援してくださる方からの 任意のご支援（ドネーション） を受け付けています。

提供プロダクト

  • KaroSpeederChrome拡張： 動画視聴を最大10倍速まで加速し、視聴時間の最適化を支援するツール
  • R-Click EraserChrome拡張： 右クリック操作を起点に、作業の手間を減らす軽量ユーティリティ

※各プロダクトの配布リンク（Chrome Web Store 等）は公開準備が整い次第掲載します。

ご支援（ドネーション）について

ご支援は 任意 です。支援の有無に関わらず、公開している機能はご利用いただけます。 いただいたご支援は、開発・検証環境の維持、アップデート作業、サポート対応（可能な範囲）に充てられます。
